Project Overview

This is a full-stack bookstore application built using Python Django, implementing the Model-View-Template (MVT) architectural pattern. The application features real-time notifications, RESTful APIs, and AI-powered book summaries.

Architecture

Backend

  • Django (Python Framework)
  • Django REST Framework
  • Django Channels

Real-time Features

  • Channel Layers
  • WebSocket Protocol

AI Integration

  • Book Summary Generation

Infrastructure

  • Render (Deployment)
  • Gunicorn (WSGI Server)

Key Features

  • Book Management: Comprehensive book catalog system
  • Order Processing: Real-time status updates (Processing → Shipped → Delivered)
  • AI-powered Summaries: Automatic generation of book summaries
  • REST API: Clean API endpoints for all operations

API Documentation

All endpoints are prefixed with /api/v1/

GET /books/

Retrieve list of all books

GET /books/{book_id}/

Get detailed information about a specific book

POST /orders/

Place a new order

GET /orders/status/

Check current order status

GET /books/summary/{book_id}/

Get AI-generated summary of a book

Deployment Details
